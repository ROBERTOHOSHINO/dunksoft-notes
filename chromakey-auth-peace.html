Concept

技術ではなく、記憶が鍵になる

従来の認証システムはパスワードや生体情報に依存します。
CHROMAKEY AUTH™ は全く異なるアプローチを取ります——

「共に体験した記憶」そのものを認証の核心に置く。

軽井沢のディナーテーブルに咲いたAnthroslanの濃いピンク。その色と香りをその場で共有した人だけが持つ記憶。それは複製も盗用もできない、生きた鍵です。

Federation 連携構造 — 問題発生時は切断、ネットワークの純度を保つ

Architecture

三層の信頼構造

LAYER 01

個人の記憶 — Personal Memory Key

色・香り・場所・時間。ある瞬間を体験した個人だけが持つ記憶のパターン。Anthroslanの濃いピンクが軽井沢の食卓に咲いた夜のように。

LAYER 02

グループ共有認証 — Community Auth

バザールバザールのメンバー同士が同じ体験を証言し合う。複数人の記憶が重なるほど認証は強固になる。一人では偽れないが、集団は共鳴する。

LAYER 03

クロスグループ連携 — Federation Network

接続した別の会とのネットワークがさらに複雑なパターンを生む。認証パターンの組み合わせが指数的に増加し、ほぼ破れない構造へ。

Peace Vision

繋がっている相手とは、戦えない

核抑止（MAD）は恐怖で戦争を止めようとします。
CHROMAKEY AUTH™ の発想は根本的に異なります。

認証ネットワークに絡んだ相手を攻撃することは、自分自身のネットワークを壊すことを意味します。破壊は自らの信頼基盤を崩す——だから誰も引き金を引けない。

信頼で戦争を無意味にする。

小さなコミュニティの食卓から積み上げて、世界の紛争がオセロのようにひっくり返る日を目指して。

「世界の紛争が、この仕組みで小さなコミュニティから積み上げて
オセロのゲームのようにひっくり返るのが、私の願い」
